Git & GitHub

Verzovanie & Kolaborácia

Git je môj "stroj času" pre kód. Umožňuje mi experimentovať bez strachu, že niečo nenávratne pokazím. GitHub je potom miesto, kde tento kód žije, kde ho zdieľam a kde môžem spolupracovať s inými (alebo so svojimi AI agentmi).

Prečo sú pre mňa kľúčové?

1. Bezpečné experimentovanie

Vďaka vetvám (branches) môžem skúšať nové funkcie alebo veľké refaktoringy bokom od hlavného kódu. Ak sa niečo nepodarí, jednoducho vetvu zahodím a hlavný projekt ostane čistý.

2. História zmien

Každý commit je ako záchytný bod. Viem presne, čo som kedy zmenil a prečo. Ak sa objaví chyba, viem sa vrátiť v čase do bodu, kedy všetko fungovalo.

3. Spolupráca s AI

Pri práci s agentmi ako Antigravity je Git nevyhnutnosťou. Agent robí zmeny, ktoré si môžem pred schválením pozrieť ako "diff". Je to základ môjho workflowu.

4. Portfólio

Môj GitHub profil slúži ako živý životopis. Ukazuje nielen to, čo som vytvoril, ale aj to, ako pracujem, ako často kódim a ako sa moje schopnosti vyvíjajú.
